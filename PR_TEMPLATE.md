## 📋 변경 사항

### 주요 변경
- **후기 작성 다이얼로그 리팩토링**: 컨테이너-프레젠테이션 패턴으로 구조 개선
  - UI 컴포넌트와 비즈니스 로직 분리
  - 커스텀 훅(`use-review-write`)으로 API 호출 로직 추상화
  - 5개의 독립적인 프레젠테이션 컴포넌트로 분리

### 컴포넌트 분리
- `ReviewHeader`: 헤더 및 닫기 버튼
- `BreederSummary`: 브리더 정보 요약
- `ReviewTabs`: 상담/입양 후기 탭
- `ReviewTextarea`: 후기 입력 영역
- `ReviewFooter`: 후기 작성 버튼

### 개선 사항
- `button-brown` 클래스를 `Button` 컴포넌트의 `primary` variant로 변경
- Tailwind spacing scale 적용 (`py-[0.5rem]` → `py-2`, `px-[1rem]` → `px-4`)
- 탭과 텍스트 에어리어 사이 간격 조정 (`gap-5` → `gap-4`)

## 🎯 목적

후기 작성 다이얼로그의 가독성과 유지보수성을 개선하기 위해 컨테이너-프레젠테이션 패턴을 적용했습니다.
