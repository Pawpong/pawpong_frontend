# 서비스 소개 배너 패드 버전 레이아웃 수정

## 변경 사항

### 서비스 소개 배너 패드 버전 레이아웃 수정
- **파일**: `src/app/(main)/_components/service-intro-banner.tsx`
- 패드 버전의 부모 div에 `relative` 추가하여 absolute positioning이 제대로 작동하도록 수정
  - 모바일 버전에는 이미 `relative`가 있었으나 패드 버전에는 누락되어 있었음
- 배너 이미지 위치를 `top-[54%]`에서 `top-[34%]`로 조정하여 패드에서 올바르게 배치되도록 개선

## 문제점
- 패드 버전에서 배너 이미지가 가운데로 올바르게 배치되지 않던 문제
- 모바일과 동일한 구조임에도 불구하고 `relative`가 없어서 `absolute` positioning이 제대로 작동하지 않음

## 해결 방법
- 패드 버전의 부모 div에 `relative` 클래스 추가
- 배너 이미지의 `top` 위치를 조정하여 패드 화면 크기에 맞게 배치

## 관련 이슈
- 서비스 소개 배너 반응형 레이아웃 버그 수정

## 테스트 체크리스트
- [ ] 패드에서 서비스 소개 배너의 강아지 이미지가 가운데로 올바르게 배치되는지 확인
- [ ] 모바일에서 배너 레이아웃이 기존과 동일하게 작동하는지 확인
- [ ] 데스크톱에서 배너 레이아웃이 기존과 동일하게 작동하는지 확인
