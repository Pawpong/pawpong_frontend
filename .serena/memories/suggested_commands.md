# Suggested Commands

## Development
- `yarn dev` — 개발 서버 실행 (Turbopack)
- `yarn build` — 프로덕션 빌드
- `yarn start` — 프로덕션 서버 실행

## Code Quality
- `yarn lint` — ESLint 실행
- `yarn format` — Prettier 포맷팅
- `yarn format:check` — Prettier 포맷팅 확인
- `yarn typecheck` — TypeScript 타입 체크 (`tsc --noEmit`)

## Testing
- `yarn test` — Vitest 테스트 실행
- `yarn test:watch` — 테스트 감시 모드
- `yarn test:ui` — 테스트 UI
- `yarn test:coverage` — 테스트 커버리지

## Storybook
- `yarn storybook` — Storybook 개발 서버 (포트 6006)
- `yarn build-storybook` — Storybook 빌드

## System (macOS/Darwin)
- `git` — 버전 관리
- `brew` — 패키지 관리 (Homebrew)
