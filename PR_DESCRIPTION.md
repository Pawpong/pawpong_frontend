# ✨ 프로필 페이지 토스트 알림 기능 추가 및 UI 개선

## 📋 개요

프로필 수정 후 사용자에게 시각적 피드백을 제공하는 토스트 알림 시스템을 추가하고, "상담 후 공개" 기능을 개선하며, 드롭다운 메뉴를 구현하고, 수정하기 버튼의 위치를 최적화하여 모든 디바이스에서 일관된 UX를 제공합니다.

## 🎯 주요 변경사항

### 🆕 토스트 알림 컴포넌트 추가

- **Radix UI Toast** 기반의 접근성 있는 토스트 시스템 구현
- Figma 디자인에 맞춘 커스텀 스타일링 적용
  - 브랜드 컬러(`#4f3b2e`) 배경
  - 연한 베이지색(`#f6f6ea`) 텍스트
  - 체크 아이콘 및 닫기 버튼 포함
  - 부드러운 애니메이션 효과

### 🔧 "상담 후 공개" 기능 개선

- **입양 비용 범위 섹션** (`profile-basic-info.tsx`)

  - "상담 후 공개하기" 버튼 클릭 시 입력된 가격 값 자동 초기화
  - 상담 모드 활성화 시 입력 필드 비활성화 및 placeholder로 "상담 후 공개" 표시
  - 상담 모드 해제 시 다시 입력 가능하도록 복원

- **분양 중인 아이 섹션** (`breeding-animals.tsx`)
  - 각 아이별로 개별 "상담 후 공개하기" 기능 적용
  - 상담 모드 전환 시 해당 아이의 입양 비용 값 초기화
  - 일관된 UX 제공을 위한 동일한 동작 구현

### 📋 드롭다운 메뉴 UI 구현

- **Radix UI DropdownMenu** 기반의 품종 선택 드롭다운 구현
- **분양 중인 아이 섹션** (`breeding-animals.tsx`)

  - 품종 선택 드롭다운: 상위에서 선택한 품종 목록을 드롭다운으로 표시
  - 입양 상태 드롭다운: "입양 가능", "입양 예정", "입양 완료" 옵션 제공
  - 엄마/아빠 선택 드롭다운: 등록된 부모 정보를 드롭다운으로 선택 가능

- **엄마 아빠 정보 섹션** (`parents-info.tsx`)
  - 품종 선택 드롭다운: 상위에서 선택한 품종 목록을 드롭다운으로 표시
  - 드롭다운 메뉴 너비 자동 조정 및 최소 너비 설정 (`min-w-[353px]`)
  - 선택된 품종이 없을 경우 안내 메시지 표시

### 📱 반응형 레이아웃 개선

- **수정하기 버튼 위치 최적화**

  - 모든 화면 크기에서 고정 위치(fixed) 적용
  - 모바일/패드: 화면 중앙, 하단에서 `1.5rem` 간격
  - 데스크톱: 왼쪽 3/4 위치, 하단에서 `2.5rem` 간격

- **토스트 위치 최적화**
  - 수정하기 버튼 위에 `0.75rem` 간격으로 배치
  - 모든 디바이스에서 버튼 위치에 맞춰 자동 조정
  - 토스트 너비는 수정하기 버튼과 일치하도록 설정

### 🔧 기술적 개선사항

- CSS 변수 파싱 오류 수정 (슬래시/괄호 포함 변수명 → 직접 색상 값 사용)
- 토스트 표시 시간 커스터마이징 가능하도록 구현
- `useToast` 훅을 통한 간편한 토스트 사용 인터페이스 제공

- 토스트 텍스트: `#f6f6ea` (연한 베이지)
- 토스트 형태: 둥근 모서리 (`rounded-full`)
- 그림자: `0px 0px 13px 0px rgba(12, 17, 29, 0.08)`

## 💡 사용 예시

```tsx
import { useToast } from "@/hooks/use-toast";

function MyComponent() {
  const { toast } = useToast();

  const handleAction = () => {
    toast({
      title: "프로필이 수정되었습니다.",
    });
  };

  return <button onClick={handleAction}>수정하기</button>;
}
```
